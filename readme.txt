1. Open terminal in project root folder
2. Use yarn dev
3. Use docker-compose up -d
4. If .env does not exist, create it and copy contents of .env.dist file into .env file
5. Go to page in browser (localhost:3000 by default)
6. Add /admin to address and click Enter
7. Upload your json file using form and then click Send
8. Enjoy backend magic that does inserting or updatading database in background
9. Once its finished, you can go back to homepage by removing /admin
10. Now everything should work, In case of problems or complaints please contact owner of this app
